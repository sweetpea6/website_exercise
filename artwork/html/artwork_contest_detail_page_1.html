제12회 자생 동·식물 세밀화 공모전

제12회 자생 동·식물 세밀화 공모전

■ 공모주제 : 「우리생물, 자세히 들여다보기」

■ 제출기간 : 2017. 8. 17(목) ~ 2017. 8. 25(금) 10:00~17:00
※ 단, 토·일요일은 현장 비접수

■ 참가대상 및 작화지침
- 성인부(대학생 포함)
(일반분야) 자생생물의 생태적 특성 묘사
※ 생태적 특성 : 라이프 사이클, 암컷과 수컷, 번식방법, 포식자와 피식자 등
(학술묘사 분야) 1개체 단독으로 작화
※ 분류군별 특징에 따른 근거자료 제출

- 학생부(초·중·고등학생)
(일반분야) 자생생물을 대상으로 작화, 생태적 특성 묘사 필수 아님

■ 공모 부문
- 일반 분야 : 성인부(대학생 포함), 고등학생, 중학생, 초등학생
- 학술묘사 분야 (Scientific drawing) : 성인부(대학생 포함)

■ 출품 시 유의 사항
- 출품은 1인 1작품, 개인 응모를 원칙으로 함
- 공모전 참가 신청서 기재 내용이 누락된 경우 작품을 접수하지 않음
- 원화(본그림) 제출을 원칙으로 함 (이미지 스캔·출력한 작품은 제외)
- 작품에 사용한 사진의 출처 또는 채집하여 관찰한 증거 자료를 반드시 제출
- 작품지 정보
* 작품 크기 : 4절지(4×6전지), 크기(394㎜×545㎜)
* 종이 재질 : 흰색 종이 필수 (두께는 제한 없음), 학술묘사 분야는 트레이싱페이퍼 종류 가능

■ 작화 기법
- 작화 기법 : 제한 없음

- 작화 배경
* 일반 분야 : 대상 생물의 표현에 방해가 되지 않는 선에서 생태적 연출을 위한 배경 작화 가능
* 학술묘사 분야 : 배경 작화 불가
- 작화 재료
* 일반 분야 : 파스텔, 목탄, 연필 등 번짐의 우려가 있는 재료 외 자유 선택(채색 자유)
* 학술묘사 분야 : 펜과 잉크만 가능(채색 불가)
- 기타 사항 : 역동적·드라마틱한 연출보다는 개체의 학술적 특성을 명확히 드러내는 작품 구도 필수
* 일반 분야 : 최소 2개체 이상 대상생물의 조화·공생 작화 필수
* 학술묘사 분야 : 1개체 작화, 스케일바, 부분도 필수
※ 부분도 : 세밀화 내 중심이 되는 그림 주변에 생물의 특징적인 부분(순비기나무 꽃의 수술, 할미꽃의 뿌리, 고라니의 송곳니 등)이나 생물의 생장과정(식물의 씨앗·떡잎, 곤충의 알·유충·번데기 등)을 알 수 있도록 1:1 비율 또는 확대하여 추가로 그린 그림
※ 스케일바 : 작품 내 생물(부분도 포함)의 크기를 가늠할 수 있도록 작품 내 정확한 치수를 배율로 표시한 막대

■ 대상 종 선정 시 유의 사항
- 재배·사육 중인 생물이나 외래종은 제외 (예: 개, 고양이, 소, 집오리, 황소개구리, 뉴트리아, 벼, 사과나무 등)
※ 두릅과 같이 재배와 야생 서식이 가능한 종은 가능함
- 학술묘사 분야 초본 작화의 경우 뿌리 등을 포함한 개체 전체 모습 작화 필수

■ 응모 자격
- 성인부(대학생 포함), 고등학생부, 중학생부, 초등학생부의 4개 부문
※ 비재학 청소년은 청소년증 등의 증빙을 갖추어 접수 가능
- 지난 공모전 2회 이상 대상, 최우수상 수상자는 심사에서 제외됨

■ 시상내역
- 성인부(대학생 포함) (총 상금 2,200만원)
* 대상(1점) : 환경부장관상 / 500만원
* 최우수상(3점) : 환경부장관상 / 각 300만원
* 우수상(5점) : 국립생물자원관장상 / 각 100만원
* 입선(6점) : 국립생물자원관장상 / 각 50만원
- 학생부(초·중·고등학생) : 총 부상 390만원
* 대상(3점) : 환경부장관상 / 문화상품권 50만원
* 최우수상(6점) : 국립생물자원관장상 / 문화상품권 각 20만원
* 우수상(12점) : 국립생물자원관장상 / 문화상품권 각 10만원
* 입선(18점) : 국립생물자원관장상 / 기념품
